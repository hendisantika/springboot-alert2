# springboot-alert2

### Things todo list:

1. Clone this repository: `git clone https://github.com/hendisantika/springboot-alert2`
2. Navigate to the folder: `cd springboot-alert2`
3. Run the application: `mvn clean spring-boot:run`
4. Open your favorite browser: http://localhost:8080

| Common parameters   |                      |                                                                                                                                                                                                                                                |
|---------------------|----------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| parameter           | Predetermined values | description                                                                                                                                                                                                                                    |
| title               | null                 | The title of the modal dialog. It can be set in the title parameter of the parameter object or in the first parameter of the swal () method.                                                                                                   |
| text                | null                 | The content of the modal dialog. It can be set in the text parameter of the parameter object, or in the second parameter of the swal () method.                                                                                                |
| html                | null                 | The content in the dialog box serves as an HTML tag. If both text and html parameters are provided, the plugin will use the text parameter first.                                                                                              |
| type                | null                 | The type of scene in the dialogue. There are 5 types of built-in scenarios: warning, error, success, information, and questions. It can be set in the type parameter of the parameter object, or in the third parameter of the swal () method. |
| customClass         | null                 | The custom class for the modal dialog.                                                                                                                                                                                                         |
| animation           | TRUE                 | If set to false, the dialog will have no animation effect.                                                                                                                                                                                     |
| allowOutsideClick   | TRUE                 | Whether to allow clicking outside the dialog to close the dialog.                                                                                                                                                                              |
| allowEscapeKey      | TRUE                 | Whether to allow the Esc key to close the dialog box.                                                                                                                                                                                          |
| showConfirmButton   | TRUE                 | If the "Confirm" button is displayed.                                                                                                                                                                                                          |
| showCancelButton    | FALSE                | If the "Cancel" button is displayed.                                                                                                                                                                                                           |
| confirmButtonText   | "OKAY"               | Confirm the text on the button.                                                                                                                                                                                                                |
| cancelButtonText    | "Cancel"             | The text on the cancel button.                                                                                                                                                                                                                 |
| confirmButtonColor  | "# 3085d6"           | Confirm the button color. It must be a HEX color value.                                                                                                                                                                                        |
| cancelButtonColor   | "#Aaa"               | The color of the cancel button. It must be a HEX color value.                                                                                                                                                                                  |
| confirmButtonClass  | null                 | Confirm the button's custom class.                                                                                                                                                                                                             |
| cancelButtonClass   | null                 | The custom class of the cancel button.                                                                                                                                                                                                         |
| buttonsStyling      | TRUE                 | Add the default swal2 style for the button. If you want to use your own button style, you can set this parameter to false.                                                                                                                     |
| reverseButtons      | FALSE                | If you want to show the button position in reverse, set this parameter to true.                                                                                                                                                                |
| showLoaderOnConfirm | FALSE                | When set to true, the button is disabled and a progress bar is displayed during loading. This parameter is used in the case of AJAX requests.                                                                                                  |
| preconfirm          | null                 | The function executed before the commit returns a Promise object.                                                                                                                                                                              |
| imageUrl            | null                 | Customize images for modal dialog boxes. URL that points to an image.                                                                                                                                                                          |
| imageWidth          | null                 | If the imageUrl parameter is set, you can set the display width and pixel unit for the image.                                                                                                                                                  |
| imageHeight         | null                 | If the imageUrl parameter is set, you can set the display height of the image in pixels.                                                                                                                                                       |
| imageClass          | null                 | Custom image class.                                                                                                                                                                                                                            |
| timer               | null                 | Timer to automatically close the dialog in milliseconds.                                                                                                                                                                                       |
| width               | 500                  | The width of the modal window, including the padding value (box-sizing: border-box).                                                                                                                                                           |
| padding             | twenty               | The fill of the modal window.                                                                                                                                                                                                                  |
| background          | "#Fff"               | The background color of the modal window.                                                                                                                                                                                                      |
| input               | null                 | The form input field type can be "text", "email", "password", "textarea", "select", "radio", "check box" and "file".                                                                                                                           |
| inputPlaceholder    | ""                   | Placeholder for the input field.                                                                                                                                                                                                               |
| inputValue          | ""                   | The initial value of the input field.                                                                                                                                                                                                          |
| inputOptions        | {} or promise        | If the input value is select or radius, you can provide options for them. The object key represents the value of the option, and the value represents the text value of the option.                                                            |
| inputAutoTrim       | TRUE                 | Whether blanks at both ends of the returned string are automatically cleared.                                                                                                                                                                  |
| inputValidator      | null                 | Whether to check the input field and return the Promise object.                                                                                                                                                                                |
| inputClass          | null                 | Custom input class.                                                                                                                                                                                                                            |

| Common methods                                |                                                                                                                                            |
|-----------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------|
| Method name                                   | description                                                                                                                                |
| swal.setDefaults ({Object})                   | When you use SweetAlert2, there are many custom parameters, you can set them as default parameters via swal.setDefaults ({Object}) method. |
| swal.resetDefaults ()                         | Restore default settings.                                                                                                                  |
| swal.queue ([Array])                          | Provide an array of SweetAlert2 parameters to display multiple dialog boxes. The dialog boxes will appear one after the other.             |
| swal.close () or swal.closeModal ()           | Close the currently open SweetAlert2 dialog box programmatically.                                                                          |
| swal.enableButtons ()                         | Confirmation and close buttons are available.                                                                                              |
| swal.disableButtons ()                        | Turn off the confirmation and close the buttons.                                                                                           |
| swal.enableLoading () or swal.showLoading ()  | Disable the button and show the loading progress bar. It is usually used for AJAX requests.                                                |
| swal.disableLoading () or swal.hideLoading () | Hide the progress bar and make the button available.                                                                                       |
| swal.clickConfirm ()                          | Click the confirm programmatically button.                                                                                                 |
| swal.clickCancel ()                           | Click the Cancel Programmatically button.                                                                                                  |
| swal.showValidationError (error)              | Show form verification error message.                                                                                                      |
| swal.resetValidationError ()                  | Hide form verification error information.                                                                                                  |
| swal.enableInput ()                           | Make the input field available.                                                                                                            |
| swal.disableInput ()                          | Disable the input field.                                                                                                                | 

See: https://sweetalert2.github.io/
